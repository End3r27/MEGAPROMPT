Corvina – Minecraft Fabric Mod 1.21.1
1. Mod Overview

Name: Corvina

Theme: Magical crows that interact with the player and world, controlled via a staff.

Core Features:

Wild and tamed crows with advanced AI

Corvina Staff for crow magic spells

Crow Essence resource system

Ritual system using vanilla candles

Shoulder crow companion system

Environment-influenced spell potency and resource availability

2. Core Gameplay Systems
2.1 Crow Entities

Types: Wild, Tamed, Shoulder Crow

Behavior:

Wild: perches, flies, collects shiny items, avoids players

Tamed: follows player, assists in spells, perches on shoulder

Drops: Feathers, rare crow eggs, shiny items

Taming: Right-click with shiny item (gold/iron nugget)

AI Goals:

Wander, escape danger, look at player, collect items, follow player

Implementation Notes:

Use AnimalEntity subclass

Shoulder logic: onShoulder flag suppresses AI, sets position relative to player head

2.2 Corvina Staff

Attributes:

Max Crow Essence: 100 units

Spells cost 10–30 essence

Power scales with nearby tamed crows

Usage:

Right-click → cast spell

Shift + Right-click → charged spell

Acquisition:

Crafting: Stick/Blaze Rod + 3 Crow Feathers + Shiny Item

Loot/Ritual reward (optional)

Enhancements:

Staff levels up with use: visual changes, new spells unlocked

Tooltip shows essence and nearby crow bonus

Implementation Notes:

Item class: CorvinaStaffItem extends Item

Track Crow Essence in NBT or capability

Use PlayerTickEvent to update essence regeneration from nearby crows

2.3 Crow Magic Spells
Spell	Type	Mechanics	Essence Cost	Crow Influence
Flock Strike	Offensive	Summons 3–5 crows to attack	20	+1 per nearby tamed crow reduces cost
Shadow Veil	Defensive	Temporary invisibility	15	Duration scales with crows
Item Snatch	Utility	Crow fetches nearby item	10	Number of crows determines max items
Ritual Empower	Support	Amplifies candle rituals	15	Buff potency scales with crows
Corvine Flight	Mobility	Short glide/jump boost	20	Crow count reduces essence cost
Dark Feather Trap	Control	Block area/scare mobs	25	Crow number increases area size

Implementation Notes:

Abstract class CrowSpell → implement each spell subclass

Spells reference player.getNearbyCrows() for scaling

Particle + sound effects triggered on cast

2.4 Crow Essence System

Resource: Crow Essence (mana)

Gained From:

Crow feathers

Crow collecting shiny items

Rituals performed

Spent On: Casting spells

Implementation Notes:

Stored in staff NBT: crow_essence

Tick-based regeneration if near tamed crows: essence += nearbyCrows * 0.1 per tick

2.5 Ritual System

Trigger: Player lights candle with staff equipped & tamed crow nearby

Effects:

Particle effects (feathers, shadows)

Buffs (luck, night vision, regeneration)

Summon temporary crows

Unlock new spells

Scaling Factors:

Number of nearby tamed crows

Number of candles lit

Time of day or weather

Implementation Notes:

Event: PlayerInteractEvent.RightClickBlock

Method: triggerCrowRitual(World world, BlockPos pos, PlayerEntity player)

Client-side: spawn particles, play sounds

2.6 Shoulder Crow Mechanics

Behavior:

Perches on player when sneaking

Follows player while moving

Assists in spells or rituals

Idle animations: flap, caw

Implementation Notes:

Override AI when perched: disable movement/goals

Update position relative to player.headPos every tick

Listen to spell casts to trigger assist animations

2.7 Environment & Game State Integration

Time & Weather:

Night: increased wild crow spawn

Rain: more shiny items found

Full moon: higher ritual potency

Game Scenario Modifiers:

PvP: configurable limits for essence cost or damage

Boss fights: increased resource drain

Implementation Notes:

boolean night = world.getTimeOfDay() >= 13000;
boolean raining = world.isRaining();
int potencyModifier = (night ? 1 : 0) + (raining ? 1 : 0);

3. Visual & Audio Design

Spells: Shadow feathers, swirling particles, staff glow

Crows: Caw sounds on idle, attack, ritual assistance

Rituals: Candle flicker, shadow swarm particles

Staff Leveling: Color shift, glowing feathers on tip

Implementation Notes:

world.addParticle() for particles

Register sound events: crow_caw.ogg, spell_cast.ogg

4. Player Feedback

Tooltips: Show essence, nearby crow bonus, available spells

Particle and sound effects indicate spell success/failure

Shoulder crow animations communicate readiness
